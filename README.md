# Wanfang Guide — aktualizacja 0.4

Ta paczka zawiera poprzednią rozbudowaną wersję materiału o **Zakazanym Mieście** oraz nowy tryb prowadzenia całego planu trasy.

## Co zmienia wersja 0.4

- pełne logo **万方** zamiast samego znaku 万,
- przycisk **Uruchom trasę** na ekranie planu,
- możliwość rozpoczęcia trybu przewodnika od dowolnego punktu planu,
- stały panel nawigacji podczas czytania,
- pasek postępu bieżącego tematu i całej trasy,
- przyciski poprzedni / następny,
- lista pozwalająca przeskoczyć do dowolnego tematu z planu,
- wyszukiwarka całej bazy dostępna bez wychodzenia z trybu przewodnika,
- układ dostosowany również do telefonu.

## Aktualizacja na GitHubie

1. Otwórz repozytorium `wanfang-guide`.
2. Kliknij **Add file → Upload files**.
3. Rozpakuj ZIP i wejdź do środka rozpakowanego folderu.
4. Przeciągnij **całą zawartość folderu** do głównego katalogu repozytorium.
5. Kliknij **Commit changes**.
6. Odczekaj 1–3 minuty i wykonaj twarde odświeżenie: `Ctrl+F5` albo `Cmd+Shift+R`.

Ulubione, plan trasy i własne notatki są zapisane w pamięci przeglądarki, więc aktualizacja plików na GitHubie ich nie usuwa.

Pełny samodzielny materiał źródłowy o Zakazanym Mieście znajduje się również w `materials/zakazane-miasto-pelny-przewodnik.html`.
